Student Command Centre is a Streamlit-based academic dashboard designed to help students manage their academic performance, tasks, and productivity in one centralized platform.

The application allows users to track attendance, monitor internal marks, manage deadlines, and visualize progress through interactive charts. It uses persistent CSV storage to maintain task and course data across sessions, making it a lightweight yet practical productivity tool for students.

Key Highlights

Academic performance tracking (attendance & marks)

Task and deadline management with automatic sorting

Progress visualization

Persistent local data storage

Clean and interactive UI using Streamlit

Tech Stack

Python

Streamlit

Pandas

This project demonstrates practical implementation of:

Data handling with Pandas

File-based persistent storage

Real-time dashboard UI development

Git-based version control and cloud deployment
